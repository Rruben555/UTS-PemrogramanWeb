JavaScript

Javascript adalah bahasa pemrograman paling populer di dunia. Digunakan untuk HTML dan web, untuk PC, Smartphone, tablet, dsb. Javascript merupakan bahasa script. Bahasa script adalah bahasa pemrograman yang ringan. Dapat dimasukkan ke dalam HTML, dan nantinya dapat dieksekusi oleh semua web browser modern.

Tailwind

TailwindCSS adalah framework CSS berbasis utility-first, yang memberikan kelas-kelas utilitas kecil untuk langsung diterapkan pada elemen HTML. Maksud dari pendekatan utility-first adalah Tailwind telah menyediakan ratusan Class CSS yang dapat kita panggil dengan mudah pada dokumen HTML, sehingga kita tidak perlu lagi melakukan penulisan CSS secara custom karena class CSS yang disediakan oleh tailwind sudah sangat lengkap. TailwindCSS lebih mengutamakan fleksibilitas dengan memberikan kontrol penuh kepada pengembang melalui kelas utilitas seperti flex, p-4, dan text-center.
